# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0-stable] - 2026-02-15

### 🎉 Initial Stable Release

这是项目的首个稳定版本，包含完整的功能实现和优化。

### ✨ 新功能

- **照片管理**
  - 响应式照片展示，适配所有设备
  - 照片上传支持拖拽
  - 客户端自动生成缩略图（400x400，质量70%）
  - 缩略图存储在独立的 `/thumbnails/` 目录
  - 照片点击查看大图
  - 照片编辑（标题、描述、相册）
  - 照片删除（同时删除原图和缩略图）

- **相册管理**
  - 创建、编辑、删除相册
  - 相册描述功能
  - 相册照片数量统计
  - 删除相册前必须清空照片

- **可视化编辑**
  - 拖拽照片到画布
  - 调整照片大小和位置
  - 自由布局编辑
  - 布局保存和恢复
  - 预览功能

- **EXIF 信息**
  - 自动提取 JPEG EXIF 信息
  - 显示相机、镜头、ISO、光圈、快门、焦距
  - 显示拍摄时间和 GPS 位置
  - 优化显示格式（ISO数字、f/光圈、焦距mm）

- **用户认证**
  - 安全的登录系统
  - 会话管理
  - 密码修改功能
  - 默认管理员账号

- **网站设置**
  - 网站标题
  - LOGO 文本
  - 作者姓名
  - 联系方式
  - 关于内容
  - 欢迎语
  - ICP 备案号

- **用户体验**
  - 中英文双语切换
  - 移动端优化
  - 侧边栏导航
  - 汉堡菜单（移动端）
  - Toast 通知
  - 加载状态显示

### 🔧 技术实现

- **架构**
  - Cloudflare Workers Serverless 架构
  - D1 SQLite 数据库
  - R2 对象存储
  - 客户端 Canvas API 缩略图生成

- **性能优化**
  - CDN 缓存
  - 数据库索引优化
  - 缩略图独立存储
  - 静态资源压缩

- **安全性**
  - bcrypt 密码加密
  - JWT 会话管理
  - API 认证中间件
  - 输入验证

### 📱 移动端优化

- 响应式布局
- 统一的左右边距
- 触摸友好的交互
- 优化的图片显示
